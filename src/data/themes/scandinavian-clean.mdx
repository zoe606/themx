Scandinavian Clean design translates the Nordic design philosophy of "beautiful things that work" into digital interfaces. Like a well-designed piece of Swedish furniture, every element serves a purpose, nothing is superfluous, and the result is an interface that feels both effortless and considered. Light, airy, and supremely functional, this style prioritizes usability without sacrificing visual appeal.

## When to Use

Scandinavian Clean is a versatile style that works across a wide range of applications -- SaaS products, e-commerce platforms, content management systems, documentation sites, and corporate websites. It is particularly effective for products that serve daily use, where visual noise would become fatiguing over time. The style scales well from simple landing pages to complex multi-feature applications, making it a safe and effective choice for most web projects.

## Key Visual Elements

The palette is dominated by white and light grays, with a single primary color (typically blue) used for interactive elements and calls to action. Borders are subtle (1px, light gray) or replaced entirely by whitespace and background color changes. Shadows, when used, are soft and minimal -- just enough to suggest elevation without creating visual weight. Padding and margins are generous, following a consistent spacing scale (8px base unit). Every pixel of space is intentional.

## Typography

Inter is the quintessential Scandinavian digital typeface -- clean, highly legible, and perfectly suited to UI text at all sizes. Use a restrained set of sizes (perhaps 14px body, 18px subheading, 24px heading, 36px hero) with consistent line heights. Font weight variations (regular for body, medium for labels, semi-bold for headings) create hierarchy without introducing additional visual complexity.
