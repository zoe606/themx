Material Design 3 (also known as Material You) represents Google's evolution from a rigid specification into a flexible, expressive design system. Introduced alongside Android 12, it emphasizes personal expression through dynamic color, adaptive layouts, and updated component designs. M3 moves away from the sharp edges and prominent shadows of earlier Material versions toward softer shapes, tonal color surfaces, and a warmer overall feel.

## When to Use

Material Design 3 is the natural choice for Android applications, cross-platform apps targeting Google's ecosystem, enterprise tools that need a well-documented component library, and any project that values accessibility and systematic design. Its comprehensive specification and extensive component library make it particularly efficient for teams that need to move fast without sacrificing quality or consistency.

## Key Visual Elements

Tonal surfaces replace the heavy drop shadows of earlier Material Design. Instead of floating cards casting dark shadows, M3 uses subtle color tinting to create elevation -- surfaces get slightly lighter or more tinted as they rise in the z-axis. Shapes are rounder: buttons use full border-radius, chips and badges are pill-shaped, and containers have generous corner rounding. The dynamic color system derives an entire palette from a single seed color, ensuring harmony across the interface.

## Typography

Roboto remains the typeface of choice, but M3 uses it with more nuance -- lighter weights, more generous line heights, and a type scale that ranges from Display (large hero text) down to Label (small UI text). The type system is organized into roles rather than fixed sizes, making it adaptable across screen sizes.
